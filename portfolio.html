Axtro-Soft Dashboard
is a Node.js library I built to make API development faster, fewer errors and better performance.
Currently it only supports Laravel framework, later it will be developed to support both CodeIgnator and Node.js.
It also designs the control panel and connects it with the API automatically without the need to write any programming commands for the design or connection.
It uses Axios library to do GET, POST, DELETE, UPDATE commands, which allows web pages to be updated asynchronously by exchanging data with a web server behind the scenes. This means that it is possible to update parts of the dashboard, without reloading the whole page..
Axtro-Soft Dashboard is in the development stage.
